ProxSweep

One command updates your whole Proxmox homelab — the host, every LXC, and Docker — in a single pass.

View on GitHub Sponsor

What it does

Stop SSH-ing into each container. ProxSweep runs one script that updates the Proxmox host, every LXC on it, and Docker — in a single sweep. Built for homelabbers who'd rather run one command than twenty.

Free and open source (MIT). Full install and usage in the README on GitHub.
